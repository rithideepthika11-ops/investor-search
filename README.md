\# AI Investor Search



\## Project Overview

This is a simple frontend application that suggests top investors based on a given sector and country.



The purpose of this project is to demonstrate basic frontend development and AI-driven logic flow.



\## Features

\- User can enter a sector (e.g., AI, FinTech)

\- User can enter a country (e.g., India, USA)

\- On clicking "Search Investors", the app displays the top 3 relevant investors with one-line reasoning



\## Tech Stack

\- HTML

\- CSS

\- JavaScript



\## AI Logic

Investor suggestions are currently mocked to simulate AI-generated responses.

This approach is used to demonstrate the AI integration flow without exposing API keys.



\## How to Run

1\. Download or clone the project

2\. Open `index.html` in any modern browser

3\. Enter sector and country

4\. Click "Search Investors"



\## Notes

This project focuses on simplicity, clarity, and learning-oriented implementation.
